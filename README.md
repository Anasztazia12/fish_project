# CARPYXA

CARPYXA is a simple three-page fishing website built with HTML and CSS. It presents carp information, general fishing content, and a short experience page in a clean and easy-to-read layout.

The goal of the project is to practice front-end fundamentals while building a small site that is structured, responsive, and visually consistent across pages.

## Strategy Plane

### Project Overview

This project is a personal fishing information website created using semantic HTML and custom CSS. The site is designed to present fishing-related content in a structured format that is easy to scan on desktop and mobile devices.

### Purpose and Value

#### Purpose of the Application

The purpose of this website is to provide a clear and accessible online space for fishing-related content. It demonstrates the use of semantic page structure, consistent styling, and responsive layout techniques.

#### Value to Users

The website gives visitors a quick way to browse carp facts, fishing information, and contact details. It is designed to be easy to navigate and comfortable to read on different screen sizes.

### User Stories

#### Must Have

- As a first-time visitor, I want to understand the website topic immediately so that I know what content it contains.
- As a user, I want to navigate between pages easily so that I can find the information I need quickly.
- As a visitor, I want to read the fish and carp content in a structured format so that the site is easy to scan.
- As a mobile user, I want the layout to adapt to my screen so that I can read it comfortably.

#### Should Have

- As a visitor, I want the layout to stay clean and readable so that I can focus on the content.
- As a returning visitor, I want the navigation to stay consistent across pages so that moving between sections feels natural.
- As a user, I want images and tables to scale properly so that the site remains usable on smaller screens.

#### Could Have

- As a visitor, I want extra visual polish such as smoother transitions or additional media so that the site feels more engaging.

## Structure Plane

### Project Structure

- `index.html` — Home page with an introduction, fish cards, a fish table, and contact details
- `carp.html` — Carp page with carp information, a table, and a map section
- `fishing-experience.html` — Short page about the relaxing side of fishing
- `assets/css/style.css` — Main stylesheet for the whole site
